# CodeQuest DOMination Assignment

This is my version of the JavaScript Fundamentals Quiz.
It features all the essential functions of a basic web quiz.
A question is asked and options for the answers are provided. When an answer is selected, they will be shown if they answered correctly or not and can no longer choose another answer until moving on to the next question. The amount of questions answered correctly will be added up and you will get your score at the end of the quiz. A restart button will also appear and you can retake the quiz as many times as you like.
